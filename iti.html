ITINERA - Travel Planner



sep 2023- may 2024


From September 2023 to May 2024, I dedicated myself to developing ITINERA, an advanced travel planning application tailored specifically for trips to Kerala. Leveraging cutting-edge technology, I focused on creating personalized and optimized itineraries to enhance user experience. The project involved building a robust backend infrastructure using JavaScript and implementing a genetic algorithm to ensure the most efficient travel routes. For the frontend, I employed HTML, CSS, Embedded JavaScript (EJS), and Bootstrap, ensuring a seamless and responsive user interface. Additionally, I harnessed publicly available datasets to enrich the application's functionality, making ITINERA a comprehensive tool for travelers.
